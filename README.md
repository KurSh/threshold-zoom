# threshold-zoom

Based off [magnus](https://github.com/stuartlangridge/magnus).

### Source

You'll need the following dependencies:

  * `gir1.2-gdkpixbuf-2.0`
  * `gir1.2-glib-2.0`
  * `gir1.2-gtk-3.0`
  * `gir1.2-keybinder-3.0`
  * `python3`
  * `python3-gi`
  * `python3-setproctitle`
